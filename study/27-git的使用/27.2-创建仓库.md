# 课时27.2
## 创建仓库
重要单词 `init=initialize 初始化` 

 1. 初始化：`git init`。
 2. 列出所有子目录 
    `windows: dir -a`。
    `unix: ls -la`    
 3. 初始化三种方式 
    -在`test1`目录下，直接`git init`
    -直接在初级目录，`git init test2`
    -直接去[https://github.com](https://github.com)拷贝，格式为
        `git clone + 仓库地址 +自定义目录名称`

 5. 其他命令：`ctrl+l`清屏。
